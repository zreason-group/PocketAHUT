# 成绩查询

## 成绩同步

成绩数据自动从教务系统同步：

1. 进入「成绩」页面
2. 默认展示**当前学期**成绩
3. 下拉刷新可手动同步最新数据

::: warning 注意
成绩仅在**教务处正式发布后**才能查看。如果教务系统已发布但 App 中未显示，请下拉刷新。
:::

## GPA 计算

系统自动计算加权 GPA，支持两种算法：

### 4.0 标准算法（默认）

| 百分制 | 绩点 | 等级 |
|--------|------|------|
| 90-100 | 4.0  | A    |
| 85-89  | 3.7  | A-   |
| 82-84  | 3.3  | B+   |
| 78-81  | 3.0  | B    |
| 75-77  | 2.7  | B-   |
| 72-74  | 2.3  | C+   |
| 68-71  | 2.0  | C    |
| 64-67  | 1.5  | C-   |
| 60-63  | 1.0  | D    |
| < 60   | 0    | F    |

可在「我的」→「设置」→「GPA 算法」切换为 5.0 算法。

## 成绩趋势

切换到「趋势」标签页，可以查看：

- **GPA 走势图**：每个学期的 GPA 变化曲线
- **学分完成进度**：已修学分 vs 总学分要求
- **各学期绩点对比**：直观看到学业变化

## 查看详情

点击任意课程，可展开查看：

- 平时成绩
- 期末成绩
- 总评成绩
- 学分和学时
- 任课老师
