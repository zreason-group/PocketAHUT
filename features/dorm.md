# 宿舍签到

晚寝签到不用再找宿管签字了，手机上就能搞定。

## 绑定宿舍

第一次使用需要绑定你的宿舍信息：

1. 进入「宿舍签到」页面，按提示绑定
2. 选择**校区**和**楼栋**
3. 输入**房间号**

绑定后系统会自动关联你的宿舍签到任务。

## 签到流程

每天到了签到时间：

1. 打开「宿舍签到」页面
2. 系统自动拉取当日签到任务
3. 点击「签到」按钮即可完成

签到需要**位置验证**，确保你确实在宿舍楼附近，防止远程代签。毕竟签到的目的是确认你安全到寝了嘛。

::: warning 注意
签到依赖 GPS 定位，请确保手机已授权位置权限。如果定位不准，走到宿舍楼门口再试试。
:::

## 签到提醒

可以设置每日签到提醒，到点自动弹通知，防止忘记签到。在「我的」→「设置」中开启。
