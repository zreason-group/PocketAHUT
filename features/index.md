# 功能概览

PocketAHUT 围绕安工大学生的日常需求，提供六大核心功能模块。点击下方卡片或左侧菜单查看详情。

## 核心功能

### [📅 课表管理](/features/schedule)

自动导入教务课表，多视图切换，上课前智能提醒。支持手动添加自定义课程。

### [📊 成绩查询](/features/grades)

学期成绩同步，GPA 自动计算，走势图分析学业变化。支持 4.0 / 5.0 两种算法。

### [💳 校园卡](/features/card)

余额查询、消费明细、在线充值、挂失解挂。月度消费统计助你合理规划开支。

### [📚 图书馆](/features/library)

馆藏检索、借阅记录、到期提醒、一键续借。再也不用跑图书馆排队。

### [🗺️ 校园导览](/features/map)

教学楼、食堂、宿舍精准标注，GPS 导航。新生入学不再迷路。

### [🔔 消息通知](/features/notifications)

考试成绩、选课提醒、校园卡消费、图书到期等通知实时推送。
