# 电费查询

宿舍还剩多少电？会不会半夜突然断电？打开手机查一下就知道了。

## 绑定宿舍

电费查询同样需要先绑定宿舍（和宿舍签到共用同一绑定信息）。在「我的」→「绑定宿舍」中设置即可。

## 查询余额

进入「电费查询」页面，会显示两项数据：

| 项目 | 说明 |
|------|------|
| 房间电量 | 照明、插座的剩余电量（度） |
| 空调用电 | 空调线路的剩余电量（度） |

::: tip 为什么分两个？
安工大宿舍的照明和空调是**分开计费**的两条线路。空调电费通常比房间电费消耗更快，夏天开空调的同学要特别留意空调余额 😅
:::

## 低电提醒

当电量低于一定阈值时，App 会推送提醒，告诉你是时候去充电费了。

## 数据来源

电量数据来自学校电费系统，实时查询，跟宿管那边看到的是一致的。
