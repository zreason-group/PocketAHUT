# 教材查询

每学期开学前都要买教材，但是哪门课用哪本书、多少钱？打开 App 查一下就知道了。

## 查看教材

1. 进入「教材查询」页面
2. 选择**学期**（默认当前学期）
3. 系统列出本学期所有课程的教材信息

每本教材显示：

| 信息 | 说明 |
|------|------|
| 课程名称 | 对应哪门课 |
| 教材名称 | 书名全称 |
| 出版社 | 哪个出版社的 |
| 作者 | 编著者 |
| ISBN | 书号（方便去网上搜二手） |

## 省钱小技巧 💰

- 拿着 ISBN 去**闲鱼、淘宝、孔夫子**搜二手，通常半价就能买到
- 学长学姐出闲置的教材也可以收，便宜又环保
- 有些课老师其实不怎么用教材，可以开学第一节课确认后再买

::: warning 注意
教材信息由教务系统提供，但实际使用情况以授课老师通知为准。有些老师会换教材或者不用教材，建议开学确认后再下手。
:::

## 数据来源

教材数据来自教务系统教材征订信息，随学期更新。
