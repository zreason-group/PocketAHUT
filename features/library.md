# 图书馆

## 馆藏检索

在「图书馆」页面顶部搜索框输入关键词，支持：

- **书名**搜索
- **作者**搜索
- **ISBN** 搜索

搜索结果展示图书的：

- 索书号（快速找书位置）
- 馆藏地点（哪个校区 / 楼层）
- 可借状态（在馆 / 已借出 / 预约中）

## 借阅记录

| 标签 | 内容 |
|------|------|
| **当前借阅** | 正在借阅中的图书列表，含到期日期 |
| **历史借阅** | 已归还的借阅记录 |

每本书显示：

- 书名和作者
- 借阅日期
- 应还日期
- 续借次数

## 到期提醒

- 图书到期前 **3 天** 推送提醒
- 到期当天再次提醒
- 如已超期，显示超期天数

## 一键续借

在「当前借阅」中找到需要续借的图书，点击 **「续借」** 按钮即可。无需到馆排队。

::: warning 续借限制
- 每本书最多续借 1 次
- 如果有人预约了该书，则无法续借
- 超期图书不能续借，需先归还
:::
