# 校园导览

## 地图浏览

打开「校园导览」，展示校园地图，包含以下地点标注：

| 类别 | 包含地点 |
|------|----------|
| 🏫 教学 | 各学院楼、公共教学楼、图书馆 |
| 🏠 宿舍 | 各宿舍楼、研究生公寓 |
| 🍔 餐饮 | 各食堂、小吃街、咖啡厅 |
| 🏥 服务 | 校医院、校园卡中心、打印店、快递点 |
| ⚽ 运动 | 操场、体育馆、篮球场、游泳馆 |

## 地点搜索

点击顶部搜索框，输入地点名称（如「三食堂」「图书馆」），即可快速定位。支持模糊搜索。

## GPS 导航

1. 在搜索结果中点击目标地点
2. 点击 **「导航」** 按钮
3. 系统显示从你当前位置到目的地的路线
4. 支持步行和骑行两种模式

## 地点详情

点击地图上的任意标记点，可以查看：

- 地点名称和分类
- 详细地址
- 开放时间（如图书馆、食堂）
- 联系电话（如校医院）

## 新生专区

开学季特别提供「新生指引」模式，高亮标注：

- 报到地点
- 各学院楼位置
- 宿舍楼分布
- 食堂和超市
