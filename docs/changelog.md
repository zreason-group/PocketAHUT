# 更新日志

## v2.0.8

<Badge type="tip" text="最新" /> <Badge type="info" text="2026-07-14" />

首次公开发布版本。

### 下载

- 📦 [PocketAHUT-v2.0.8.apk](https://github.com/zreason-group/PocketAHUT/releases/download/v2.0.8/PocketAHUT-v2.0.8.apk) — Android 安装包

### 更新内容

- 课表管理：一键导入教务课表，支持自定义课程
- 成绩查询：成绩实时同步，GPA 自动计算
- 考试安排：期末考、补考安排一目了然
- 宿舍签到：GPS 定位签到
- 电费查询：照明 & 空调独立线路余额查询
- 空教室查询：按教学楼和时间段查询空闲教室
- 洗衣机查询：空闲状态实时呈现
- 食堂菜品：全校比价、窗口评分排行
- 校园新闻：通知公告与校园动态
- 体测计算器：输入成绩秒算分数和等级

### 支持平台

- Android（APK 安装包）
- 微信小程序
