## 成为贡献者

口袋安小工 是由安工大各个年级的学生自发组成的项目，欢迎任何形式的贡献！

- **提交代码**：Fork 项目后提交 Pull Request
- **反馈问题**：在 [GitHub Issues](https://github.com/zreason-group/PocketAHUT/issues) 提交 Bug 或建议
- **完善文档**：点击每页底部的「在 GitHub 上编辑此页」直接修改
