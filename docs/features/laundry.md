# 洗衣机查询

抱着一桶衣服下楼，结果发现所有洗衣机都在转——这种绝望我们都懂。

## 查看洗衣机状态

进入「洗衣机」页面，系统会根据你绑定的宿舍楼栋，自动展示该楼的洗衣机列表：

| 信息 | 说明 |
|------|------|
| 设备编号 | 每台洗衣机的唯一标识 |
| 当前状态 | 空闲 / 使用中 / 故障 |
| 剩余时间 | 如果正在使用，显示还剩几分钟 |

一目了然，哪台能用就去哪台。

## 使用建议

- 去洗之前先打开看一眼，别白跑一趟
- 高峰期（周末晚上）洗衣机比较抢手，建议错峰
- 看到有故障标记的洗衣机直接跳过，别浪费时间试

::: tip 小窍门
洗完衣服设个闹钟，及时取走，方便下一位同学。公共设施靠大家 🙏
:::
