# 成绩查询

## 成绩同步

成绩数据自动从教务系统同步：

1. 进入「成绩」页面
2. 默认展示**当前学期**成绩
3. 下拉刷新可手动同步最新数据

::: warning 注意
成绩仅在**教务处正式发布后**才能查看。如果教务系统已发布但 App 中未显示，请下拉刷新。
:::

## 学分绩计算

学年平均学分绩=∑（课程学期成绩×课程学分）/∑课程学分。
课程学期成绩=考试成绩×对应占比+平时成绩×对应占比
课程学分绩点=该课程学分x绩点

### 考核成绩与绩点的折算

| 百分制    | 绩点  |   等级   |
| ------ | --- | :----: |
| 96-100 | 4.0 | A+（优+） |
| 90-94  | 3.7 |  A（优）  |
| 85-89  | 3.3 | B+（良+） |
| 80-84  | 3.0 |  B（良）  |
| 75-79  | 2.7 | C+（中+） |
| 70-74  | 2.3 |  C（中）  |
| 60-69  | 2.0 | D（及格）  |
| <60    | 1.5 | F（不及格） |

### 五级分制与百分制成绩的折算


| 五级分制成绩 | 百分制成绩 |
| :----: | :---: |
|   优    |  95   |
|   良    |  85   |
|   中    |  75   |
|   及格   |  65   |
|  不及格   |  55   |

## 查看全部成绩

1. 点开成绩查询默认展示**当前学期**成绩
2. 页面最上方搜索按键可以查看目标课程的成绩
3. 页面左上角可选择查看不同学期的成绩
4. 页面右上角可以查看目标课程类型
5. 最上方[奖杯]样式的图标点开可查看平均学分绩、专业排名、专业总人数

## 查看详情

点击任意课程，可展开查看：

- 平时成绩
- 期末成绩
- 总评成绩
- 成绩构成占比
- 学分和绩点
- （实验成绩）

## 成绩更新提醒

在**我的->高级功能**页面里面可打开**成绩更新提醒**（需绑定邮箱），每4小时自动检查，直接给邮箱发送通知。
