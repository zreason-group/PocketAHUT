# 空教室查询

想找个安静的地方自习，结果跑遍整栋楼发现间间都有课？这个功能就是为你准备的。

## 查询空教室

1. 进入「空教室」页面
2. 选择**校区**（本部 / 东区）
3. 选择**教学楼**（比如教一、教三）
4. 选择**日期**和**节次**

系统会列出当前条件下所有没课的教室，每条结果显示教室名和座位数。

## 时间段说明

| 时间段    | 对应节次   |
| ------ | ------ |
| 1-2 节  | 上午第一二节 |
| 3-4 节  | 上午第三四节 |
| 5-6 节  | 下午第一二节 |
| 7-8 节  | 下午第三四节 |
| 1-4 节  | 整个上午   |
| 5-8 节  | 整个下午   |
| 9-11 节 | 晚上     |
| 1-11节  | 全天     |

::: tip 实用技巧
考试周图书馆爆满？用空教室查询找个没课的教室复习，比图书馆还安静。而且可以按座位数筛选，小组讨论也能找到大教室。
:::

## 数据来源

空教室数据基于教务系统**排课信息**实时计算，准确度很高。不过偶尔调课、补课的情况可能不会立刻反映，以实际情况为准。
