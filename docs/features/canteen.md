# 食堂菜品

干饭人必备功能——看看各个食堂窗口卖什么、多少钱、评价怎么样。

## 菜品查询

进入「菜品查询」页面，可以按条件筛选：

| 筛选条件 | 说明 |
|----------|------|
| 校区 | 本部 or 东区 |
| 食堂 | 具体哪个食堂 |
| 窗口 | 具体哪个窗口 |
| 分类 | 面食、荤菜、素菜等 |
| 价格区间 | 设置最低和最高价 |

每条菜品显示名称、窗口、价格，帮你快速锁定想吃的。

## 菜品比价

同一个菜，不同食堂不同价？比价功能帮你找最实惠的：

1. 选择菜品分类（如「拌面」）
2. 输入菜品关键词（如「鸡腿」）
3. 系统会列出**全校各食堂**这个菜的价格对比

比价结果还会统计**最高价、最低价**

## 窗口评论

觉得哪个窗口好吃或者踩雷了？给它评论：

1. 进入「社区」页面
2. ~~畅所欲言~~
3. 欢迎吃饭焚决
4. 在菜谱页面可以选择窗口进行评论（右下角），也可以看指定窗口的评价
5. 社区默认随机10条评论，下拉和右下角按钮可以刷新



::: tip 注意
评论仅用于参考，不代表真实评价 ，管理员会审核，审核后才会显示
:::

## 数据来源

菜品数据由管理员维护，定期更新。如果你发现有窗口关门了或者涨价了，欢迎反馈。
