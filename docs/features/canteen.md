# 食堂菜品

干饭人必备功能——看看各个食堂窗口卖什么、多少钱、评价怎么样。

## 菜品查询

进入「菜品查询」页面，可以按条件筛选：

| 筛选条件 | 说明 |
|----------|------|
| 校区 | 本部 or 东区 |
| 食堂 | 具体哪个食堂 |
| 窗口 | 具体哪个窗口 |
| 分类 | 面食、荤菜、素菜等 |
| 价格区间 | 设置最低和最高价 |

每条菜品显示名称、窗口、价格，帮你快速锁定想吃的。

## 菜品比价

同一个菜，不同食堂不同价？比价功能帮你找最实惠的：

1. 选择菜品分类（如「荤菜」）
2. 输入菜品关键词（如「鸡腿」）
3. 系统会列出**全校各食堂**这个菜的价格对比

比价结果还会统计**均价、最高价、最低价**，哪个食堂最坑一目了然 👀

## 窗口打分

觉得哪个窗口好吃或者踩雷了？给它打分：

1. 进入「窗口评论」页面
2. 系统每天给你 **10 组窗口对比**
3. 每组两个窗口，分别打分（总分 10 分）
4. 分数越高 = 越推荐

所有同学的打分汇总后会生成**窗口排行榜**，看看全校同学心中的食堂 Top 榜。

::: tip 关于打分
打分规则很简单：10 分总分分给两个窗口，比如你觉得 A 窗口比 B 窗口好吃很多，可以打 A:8 分 B:2 分。如果差不多就各 5 分。这样比单纯的打星更真实。
:::

## 数据来源

菜品数据由管理员维护，定期更新。如果你发现有窗口关门了或者涨价了，欢迎反馈。
