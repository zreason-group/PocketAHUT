# 考试安排

期末季最怕啥？不知道哪天考哪门、在哪个教室。有了考试安排，教务系统一发布你就能看到。

## 查看考试安排

1. 首页可直接查看**最近考试**
2. 进入「考试安排」页面，需提前绑定教务系统
3. 默认显示**当前学期**所有考试

每场考试会显示：

| 信息   | 说明       |
| ---- | -------- |
| 课程名称 | 哪门课的考试   |
| 考试时间 | 具体到几点到几点 |
| 考场   | 在哪个教室考   |
| 座位号  | 你的座位编号   |

::: tip 小贴士
~~别忘了看老师叫什么~~
考前一天去看看考场位置，免得到时候满教学楼找教室。尤其是大一新生，有些教学楼布局确实挺迷的 😅
:::

## 数据来源

考试安排数据直接从**教务系统**同步，教务处发布后就能看到。如果发现某门课没显示，可能是老师还没录入，耐心等等就好。
