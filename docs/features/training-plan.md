# 培养方案

不知道自己这专业要修多少学分、有哪些必修课？培养方案帮你理清楚。

## 查看培养方案

1. 进入「培养方案」页面
2. 系统自动从教务系统拉取你所在专业的培养方案
3. 展示完整的**课程体系**和**学分要求**

内容包括：

- **公共必修课**：马原、英语、体育这些全校都要上的
- **专业必修课**：你专业最核心的课程
- **专业选修课**：可以按兴趣选的课
- **实践环节**：实习、毕业设计等

每门课标注学分、学时、开课学期。

## 有什么用？

- **规划选课**：看看还有哪些选修课学分没修够
- **毕业自查**：对着培养方案算算自己学分够不够毕业
- **转专业参考**：想转专业的话，先看看目标专业要学什么

::: tip 注意
培养方案不同年级可能不同，以你入学年份的方案为准。如果发现 App 显示的和教务处不一致，以下发到手的纸质版为准。
:::

## 数据来源

培养方案数据来自教务系统，教务处更新后 App 会自动同步。
