# 校园新闻

教务通知、学校公告、要闻动态——不用打开学校官网，App 里就能看。

## 新闻分类

新闻分为三大类：

| 分类 | 内容 |
|------|------|
| 学业通知 | 选课、考试、成绩、学籍相关通知 |
| 公告通知 | 学校行政通知、活动公告 |
| 学校要闻 | 校园新闻动态 |



## 阅读新闻

1. 点击新闻卡片进入详情页
2. 正文完整展示，跟学校官网发布的一致
3. 附件要去官网下

## 数据来源

新闻内容从学校官网**实时同步**。


::: tip 建议
把「学业通知」分类常看看，选课通知、补考安排这些错过了真的很麻烦。
:::

::: tip 注意
打开页面无显示？检查有无绑定智慧校园 [账号说明](/guide/account) 。
:::
