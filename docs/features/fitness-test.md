# 体测计算器

体测完了不知道分数怎么算的？输入你的成绩，秒出分数和等级

我们做了每个分值的占比以及加权分计算，比市面上绝大多数的体测计算器都要完善

## 计算体测成绩

1. 进入「体测计算器」页面
2. 输入你的各项体测成绩：

| 项目 | 说明 |
|------|------|
| 身高 / 体重 | 计算 BMI |
| 肺活量 | 吹气的那个 |
| 50 米跑 | 短跑 |
| 坐位体前屈 | 柔韧性 |
| 立定跳远 | 弹跳力 |
| 引体向上（男）/ 仰卧起坐（女） | 上肢/腹部力量 |
| 1000 米（男）/ 800 米（女） | 耐力跑 |

3. 点击计算，系统按**国家标准**自动算出每项得分和加权总分
4. 显示最终等级：优秀 / 良好 / 及格 / 不及格

## 评分标准

采用《国家学生体质健康标准》评分表，按**性别**和**年级**区分标准。输入时注意选择正确的性别，不然分数会算错。

::: info 说明
体测计算器是纯本地计算，不需要联网，也不会上传你的体测数据。安心用。
