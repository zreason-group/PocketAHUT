# 课表管理

## 自动导入

从教务系统一键导入课程表，无需手动录入。

1. 进入「课表」页面
2. 点击左上角「设置」按钮
3. 选择当前学期，学期开始日期，作息时间和是否显示教师名
4.点击导入课程表，系统自动拉取教务课表数据

## 多视图切换

课表页面支持三种视图模式，通过顶部标签切换：

| 视图 | 适用场景 |
|------|----------|
| **周视图** | 点进课程表查看一周完整课程安排，默认视图 |
| **日视图** | 首页聚焦今日和明日课程，快速了解以便于安排 |

## 课表查看

1.课程表最上方左右有“上一周”/“下一周”的切换按钮
2.课程表最上方中间有周数和日期显示，点击可切换年月日
