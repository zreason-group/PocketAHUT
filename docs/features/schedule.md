# 课表管理

## 自动导入

从教务系统一键导入课程表，无需手动录入。

1. 进入「课表」页面
2. 点击右上角 **「导入」**
3. 选择当前学期
4. 系统自动拉取教务课表数据

导入后，你也可以手动对课表进行调整，手动调整的部分不会被自动覆盖。

## 多视图切换

课表页面支持三种视图模式，通过顶部标签切换：

| 视图 | 适用场景 |
|------|----------|
| **周视图** | 查看一周完整课程安排，默认视图 |
| **日视图** | 聚焦今日课程，快速了解当天安排 |
| **列表视图** | 按时间线列出所有课程，适合学期总览 |

## 自定义课程

除了教务课表，你还可以添加个人安排：

1. 在课表页点击右下角 **「+」**
2. 填写课程名称、时间、地点、周次
3. 点击保存

自定义课程会以不同颜色显示，方便区分。

::: tip 用途推荐
可以用自定义课程记录实验课、社团活动、自习安排等非教务课程。
:::

## 智能提醒

| 提醒类型 | 时机 |
|----------|------|
| 上课提醒 | 课程开始前 15 分钟 |
| 考试提醒 | 考试前 1 天 + 1 小时 |
| 调课通知 | 教务发布调课信息后实时推送 |
| 停课通知 | 教务发布停课信息后实时推送 |

提醒可在「我的」→「设置」→「通知设置」中自定义开关。

## 分享课表

点击课表页右上角「分享」，可生成课表图片或链接，方便发送给同学。
