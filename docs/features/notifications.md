# 消息通知

::: warning 🚧 功能开发中
消息推送功能还在开发完善中，暂未完整上线。以下为规划内容，实际功能以上线版本为准。
:::

口袋安小工 会实时推送与你相关的各类通知。

## 通知类型

### 教务通知

| 通知类型     | 触发时机                 |
| ------------ | ------------------------ |
| 考试成绩发布 | 教务处录入成绩后         |
| 选课提醒     | 选课开始前 1 天 + 1 小时 |
| 补考安排     | 补考时间发布后           |
| 四六级报名   | 报名开始通知             |
| 调课 / 停课  | 教务处发布后实时推送     |

### 生活通知

| 通知类型       | 触发时机             |
| -------------- | -------------------- |
| 校园卡消费异常 | 单笔消费超过设定阈值 |
| 校园卡余额不足 | 余额低于设定阈值     |
| 图书到期提醒   | 到期前 3 天 + 当天   |
| 寝室电费余额   | 电费低于设定阈值     |

## 通知设置

在「我的」→「设置」→「通知设置」中可以自定义：

- **全局开关**：一键开启 / 关闭所有通知
- **分类开关**：单独控制教务通知 / 生活通知
- **免打扰时段**：设置夜间免打扰时间
- **消费提醒阈值**：自定义金额警戒线

## 推送渠道

| 渠道           | 说明                     |
| -------------- | ------------------------ |
| App 推送       | 系统通知栏推送（需授权） |
| 小程序服务通知 | 微信服务通知（需订阅）   |
| App 内消息     | 消息中心可查看历史通知   |

::: tip 消息中心
即使错过了推送通知，也可以在 App「我的」→「消息中心」查看最近 30 天的所有通知记录。
:::
