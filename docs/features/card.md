# 校园卡

::: warning 🚧 功能开发中
校园卡模块还在开发中，暂未上线。以下为规划内容，实际功能以上线版本为准。
:::

## 余额查询

进入「校园卡」页面，顶部卡片显示：

- **当前余额**：实时更新
- **今日消费**：当天累计消费金额
- **月消费**：本月累计消费金额

下拉刷新即可获取最新数据。

## 消费记录

点击「消费记录」可查看详细流水：

| 信息 | 说明 |
|------|------|
| 消费时间 | 精确到秒 |
| 消费地点 | 食堂 / 超市 / 浴室等 |
| 消费金额 | 单笔金额 |
| 余额变化 | 消费后余额 |

支持按日期范围和消费地点筛选。

## 在线充值

1. 在校园卡页面点击 **「充值」**
2. 选择充值金额（预设 + 自定义）
3. 确认后跳转微信支付
4. 支付成功后余额**实时更新**

::: tip 到账时间
微信支付成功后，余额通常实时到账。如未到账，等待 5 分钟后下拉刷新。若仍未到账，请联系校园卡中心。
:::

## 挂失与解挂

卡片丢失时的处理流程：

1. 点击校园卡页面的 **「挂失」**
2. 输入校园卡密码确认
3. 挂失成功后卡片即刻冻结，防止被盗刷

找到卡片后：

1. 点击 **「解挂」**
2. 输入校园卡密码确认
3. 解挂后卡片恢复正常使用

::: warning 注意
挂失后如需补办新卡，请前往校园卡中心办理，App 暂不支持在线补办。
:::

## 月度统计

点击「月度统计」可查看本月消费饼图，按食堂、超市、浴室等分类统计，帮你了解钱都花哪了。
