# 系统账号说明

口袋安小工可绑定安工大校园侧的多个系统账号，用于整合课表、成绩、宿舍用电、校园通知等能力。本页用于说明各个系统账号分别对应什么、在哪里使用，以及常见的密码来源。

::: warning 安全提醒 ⚠️
请勿在任何系统中使用弱密码或已泄露的密码。若无法确认初始密码，请以学校官方通知、迎新材料或对应系统提示为准。
:::

## 账号总览

| 系统     | 通常对应的账号         | 通常对应的密码     | 主要用途                             |
| -------- | ---------------------- | ------------------ | ------------------------------------ |
| 教务系统 | 9 位学号               | 8 位生日           | 课表、成绩、考试、培养方案等教务信息 |
| 宿舍系统 | 通常为学号             | `Ahgydx@920`       | 宿舍用电、电费余额等生活服务信息     |
| 智慧校园 | 通常为学校统一身份账号 | 以当年通知信息为准 | 通知公告等校园服务能力               |

## 教务系统

教务系统通常用于查询和同步与教学相关的信息，例如课表、成绩、考试安排、培养方案等。学校教务系统的网站为：[安徽工业大学教务系统](http://jwxt.ahut.edu.cn/jsxsd)

::: warning 警告 ⚠️
学校教务系统未启用 TLS 加密，因此口袋安小工服务器与学校教务系统服务器通信时，链路中可能存在被中间人攻击、账号密码被窃听或篡改的风险。该风险来自学校系统本身的连接方式，我们无法单方面消除，除非学校对教务系统启用 HTTPS / TLS 加密。继续绑定和使用教务系统能力，即表示你已了解并接受该风险。

即使你不通过口袋安小工访问教务系统，而是直接使用浏览器访问学校教务系统，由于学校教务系统本身同样未使用 TLS 加密，风险也近乎相等。请不要在不可信的网络环境内直接使用浏览器登录教务系统，例如酒吧、咖啡厅、机场、火车站等公共 Wi-Fi。
:::

### 初始账号与密码

教务系统账号为你的 **9 位学号**，初始密码通常为你的 **8 位生日**，格式为 `YYYYMMDD`。

例如：

| 信息     | 示例                 |
| -------- | -------------------- |
| 学号     | `259011001`          |
| 生日     | `2008 年 7 月 11 日` |
| 初始密码 | `20080711`           |

## 智慧校园

智慧校园通常用于承载学校统一服务能力，例如校园通知、公告、统一认证入口等。

### 初始账号与密码

智慧校园账号通常为学校统一身份相关账号，可能是学号、统一身份认证账号或学校通知中指定的账号形式。

智慧校园初始密码需要结合当年学校通知信息确认。不同入学年份、不同批次的初始密码规则可能不同，请以学校官方通知、迎新材料或智慧校园系统提示为准。

例如：

| 信息     | 示例               |
| -------- | ------------------ |
| 账号     | 以当年通知信息为准 |
| 初始密码 | 以当年通知信息为准 |
