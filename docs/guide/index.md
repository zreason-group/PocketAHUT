# 快速开始

欢迎使用口袋安小工！跟着以下步骤，3 分钟完成上手，享受 **一站式** 安工大校园信息平台，让您摆脱繁复卡顿的校园系统！

## 第一步：获取应用

口袋安小工 目前支持以下平台：

- **Android**：前往 [下载安装](/guide/install#android) 页面查看apk安装包下载方式。
- **微信小程序**：打开微信，进入「发现」→「小程序」，搜索「口袋安小工」，点击进入即可使用（**无需下载安装**）。

> iOS 和网页端的开发工作正在进行中，预计不久就可以和小伙伴们见面啦！

::: warning 警告
我们预计将会逐步停止小程序的开发和维护，请新用户优先选择Android 客户端。
:::

## 第二步：注册与登录

1. 打开 口袋安小工 应用，进入登录页
2. 点击注册页面进行 口袋安小工 账号注册
3. 输入想要注册的用户名称并设置密码，用户注册需绑定邮箱并接收验证码，以便找回密码
4. 使用设置的用户名与密码登录 口袋安小工

## 第三步：学校侧系统绑定

口袋安小工 目前可以绑定学校的以下系统并整合其提供的能力：

| 系统     | 可整合能力                           |
| -------- | ------------------------------------ |
| 教务系统 | 课表、成绩、考试、培养方案等教务信息 |
| 宿舍系统 | 宿舍用电、电费余额等生活服务信息     |
| 智慧校园 | 通知公告等校园服务能力               |

::: warning 注意 ⚠️
口袋安小工 将以先进的可逆加密算法存储你的教务系统账号密码。开发团队将尽一切可能保障用户密码和数据安全，请勿在口袋安小工及学校系统，尤其是教务系统使用弱密码或已泄露的密码，保障账号安全
:::
