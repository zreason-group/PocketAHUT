# 下载安装

## 版本说明

口袋安小工 目前主要维护 Android 客户端和微信小程序两个版本，功能将逐步向独立移动客户端集中维护。

| 平台       | 获取方式                        | 推荐程度 |
| ---------- | ------------------------------- | -------- |
| Android    | GitHub Releases 下载 APK 安装包 | 推荐     |
| 微信小程序 | 微信内搜索「口袋安小工」        | 可用     |
| 网页端     | 正在开发中，暂未开放正式版本    | 敬请期待 |
| iOS        | 正在开发中，暂未开放正式版本    | 敬请期待 |

::: warning 警告 ⚠️
我们预计将会逐步停止小程序的开发和维护，请新用户优先选择 Android 客户端。
:::

## Android

Android 客户端是目前推荐的新用户使用方式，请前往 [GitHub Releases](https://github.com/zreason-group/PocketAHUT/releases/) 下载最新版本 APK 安装包。

1. 打开 [口袋安小工 Releases](https://github.com/zreason-group/PocketAHUT/releases/)
2. 选择最新版本（Latest）
3. 在 Assets 中下载 `.apk` 安装包
4. 下载完成后，打开安装包
5. 按照系统提示允许安装并完成安装
6. 安装完成后，打开「口袋安小工」并登录使用

::: tip 安装提示
如果系统提示“禁止安装未知来源应用”，请在系统设置中允许当前浏览器或文件管理器安装应用，然后重新打开 APK 安装包。
:::

::: warning 安全提醒
请从官方 [GitHub Releases](https://github.com/zreason-group/PocketAHUT/releases/) 获取安装包，不要安装来路不明的 APK 文件，避免信息泄露。
:::

## 微信小程序

小程序无需下载，适合临时或轻量使用：

1. 打开微信，点击底部的「发现」
2. 选择「小程序」
3. 点击顶部搜索框，输入「口袋安小工」
4. 点击搜索结果进入

::: warning 维护提醒
微信小程序后续可能逐步停止开发和维护，建议新用户优先安装 Android 客户端。
:::

## iOS

iOS 客户端通过 GitHub Releases 分发 IPA 安装包，请前往 [PocketAHUT Releases](https://github.com/zreason-group/PocketAHUT/releases/) 下载最新版本。

1. 打开 [口袋安小工 Releases](https://github.com/zreason-group/PocketAHUT/releases/)
2. 选择最新版本（Latest）
3. 在 Assets 中下载 `.ipa` 安装包
4. 按照你使用的安装工具完成安装

::: warning 安装提醒
IPA 安装通常需要配合签名工具或其他受信任的安装方式，请确保安装来源可靠。
:::
