# 下载安装

## 版本说明

PocketAHUT 同步维护三个平台版本，功能一致，数据互通。

| 平台 | 下载方式 | 版本 |
|------|----------|------|
| iOS | [App Store](https://apps.apple.com) | 1.0.0 |
| Android | 华为 / 小米 / OPPO / 应用宝 | 1.0.0 |
| 微信小程序 | 微信内搜索「PocketAHUT」 | 1.0.0 |

## iOS

1. 打开 iPhone 上的 **App Store**
2. 点击底部的「搜索」
3. 输入 **PocketAHUT**
4. 点击「获取」按钮，使用面容 ID / 触控 ID 或密码确认下载

::: tip 系统要求
iOS 14.0 或更高版本。支持 iPhone 和 iPad。
:::

## Android

根据你的手机品牌，在对应应用商店下载：

| 手机品牌 | 应用商店 |
|----------|----------|
| 华为 | 华为应用市场 |
| 小米 / Redmi | 小米应用商店 |
| OPPO / 一加 | OPPO 软件商店 |
| vivo / iQOO | vivo 应用商店 |
| 其他品牌 | 应用宝 |

直接在商店搜索「PocketAHUT」即可找到。

::: warning 安全提醒
请从**官方应用商店**下载，不要安装来路不明的 APK 文件，避免信息泄露。
:::

## 微信小程序

小程序是最轻量的使用方式，无需下载、即用即走：

1. 打开微信，点击底部的「发现」
2. 选择「小程序」
3. 点击顶部搜索框，输入「PocketAHUT」
4. 点击搜索结果进入

::: tip 添加到桌面
进入小程序后，点击右上角「···」→「添加到桌面」，下次从桌面直接打开，跟 App 体验一样方便。
:::
