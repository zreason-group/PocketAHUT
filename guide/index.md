# 快速开始

欢迎使用 PocketAHUT！跟着以下步骤，3 分钟完成上手。

## 第一步：获取应用

PocketAHUT 支持以下平台，选择你常用的即可：

::: code-group

```md [iOS]
1. 打开 **App Store**
2. 搜索「PocketAHUT」
3. 点击下载安装
```

```md [Android]
1. 打开应用商店（华为 / 小米 / OPPO / 应用宝）
2. 搜索「PocketAHUT」
3. 点击下载安装
```

```md [微信小程序]
1. 打开微信 →「发现」→「小程序」
2. 搜索「PocketAHUT」
3. 点击进入即可使用（**无需下载安装**）
```

:::

::: tip 推荐
如果你不想下载 App，微信小程序是最便捷的选择，功能与 App 完全一致。
:::

## 第二步：登录

1. 打开 PocketAHUT，进入登录页
2. 输入你的**学号**和**教务系统密码**
3. 首次登录需绑定手机号（用于后续找回密码）

::: warning 注意
PocketAHUT 不会存储你的教务系统明文密码。登录走教务处官方验证，你的密码只在教务处服务器校验。
:::

## 第三步：导入课表

登录后首页为空，需要先导入课表：

1. 点击底部「课表」进入课表页
2. 点击右上角 **「导入」** 按钮
3. 选择当前学期，系统自动从教务系统拉取

课表导入后，首页将展示当日课程和待办事项。
